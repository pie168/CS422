Please follow are the sketches for the project


Overview

 
	-We have the presets options on the left side of the screen.
	-The middle consist of both the display as well as the locations of where
		the additional toast option would be. 
	-The display contains buttons on the top and bottom to allow the user
		to change time or temp depending on the position of the digits.
	-On the right bottom, we have the settings buttons.
		-Here you'll be able to set the time, temp, C/F, and start or stop 
			the process.
	-On the top right are other features for the toaster oven.
		1)Power button
		2)Lock button
		3)Sound button
		4)UI Button
	-Will go into more detail later.

Presets

	Here we have 4 presets that the user can choose from. Toast will allow the user
	to toast their bread. Selecting this will open 3 more buttons that will ask how 
	well you want your bread toasted. The following are the options for toast:
		1)Golden
		2)Crispy
		3)Burnt
	When you click on any of the preset button, the display in the middle will show you
	information regarding that preset such as time and temp.

Body

	The body of the toaster oven consist of the display and a few buttons on both 
	the top and bottom. The display will give out information such as the time and temp.
	When changing the time or temp, the numbers will be aligned so that the buttons on the 
	top and bottom will allow you to change those specific digits.

Switches

	Here we have the options to set the temp, time, and converstion of temp. We also have a
	start and stop button for obvious reasons.

Options

	The top buttons allow the user to use some extra features. The following: 
		1)Lock Button
			This locks the entire screen so after you start the toaster oven, you can't press 
			anything else other than to unlock the screen. This is to prevent any accidental 
			presses.
		2)Sound Button
			A feature this toaster oven has is that it plays sounds. During the event that you
			want a quiet setting, this will allow you to mute your toaster oven to prevent it from
			making any noise.
		3)Incognito Button
			This button hides most of the UI except the display to give you a less cluttered view on 
			what ever you're cooking.
